# 上海一模二模资料库

一个面向上海高中生的免费试卷资料检索平台。

项目目前处于早期开发阶段，计划按年份、地区、年级、科目和考试类型整理上海各区的一模、二模及其他考试资料。

## 当前功能

- 关键词搜索
- 年份筛选
- 上海各区筛选
- 年级筛选
- 科目筛选
- 一模、二模、期中、期末筛选
- 在线查看试卷与答案
- 一键清空筛选条件
- 显示搜索结果数量
